System Documentation

Upon opening the Order To Go application, there awaits a Welcome Screen that allows a user to log in if they are already a member by entering his/her U.W.I Student ID number along with their respective password and proceed to the Restaurant Screen by clicking the ‘login’ button. Non- members are given the opportunity to sign up by clicking the ‘sign up’ button which navigates them to the Sign Up Screen. Within this screen the user enters his/ her name, student id no. and password. After the user clicks the ‘submit’ button and the information is submitted and saved. By doing this, it takes the user back to the Welcome Screen where the user can log in using their credentials.

In the Restaurant Screen the user is provided with a list of restaurants located around UWI campus. The user selects a restaurant and is navigated to the Menu Screen. The user can click the drop down menu to view various sections listed such as individual items, sides and even drinks. When the user click any of the orders they desire, they are navigated to the Customize Screen. In this screen, the order that the user selects is displayed at the top of the screen along with a toggle to select and deselect their preference meal choice. Below, the user can select their various side choices that are available and also drink choices. The drop down menu allows the user to select their preference choice pertaining to the various options given. The price is displayed along with the quantity the user requires where the user can select the amount of orders they want for this specific meal. By selecting the ‘make favourite’ button, the user can make this specific order a favourite order where they can select this particular order for future purchases. Then the user proceeds to click the ‘add to cart’ button which navigates the user to the Cart Display Screen.

The user can view the orders they made including every detail pertaining to the order such as side and drink choices. The estimated time the order will take to finish is displayed along with the total cost of the specific order. The user can select the drop down menu to view the type of payments available. The user can either select “credit card” or “cash” payments. For the credit card payment option, the user is prompted to enter his/her credit card number. The red ‘X’ at the top right hand corner of the screen symbolizes cancellation of the order. The user can click the ‘X’ button to cancel their order however a certain timeframe to do so is given. Finally, the user can click the ‘confirm order’ button to purchase their specific order and a message is displayed to the user. This message states that the order is confirmed and the information is sent to the restaurant for preparation.
