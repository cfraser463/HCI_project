Heuristic Analysis

Description

In this heuristic analysis the core task of ordering a meal using the OrderToGo App was evaluated. The evaluator moved through 5 screens where they selected a restaurant, choose their desired order, customizing/ finalizing the order and placing the order.

Major Problems Found

Unable to jump directly to start menu
Violates: “User Control and Freedom “
Fix: Add navigation menu tab
Severity - 4

Difficult to find restaurants manually
Violates: Recognition rather than recall
Fix: implementing a search bar would make it easier to locate restaurant names
Severity - 3

Dropdown tabs are barely visible as they have no borders, making it hard to see when there are multiple options
Violates: “User Control and Freedom “
Fix: Have drop down tabs bordered and coloured
Severity - 3

Severity Rating

0 - don’t agree that this is a usability problem
1 - cosmetic problem
2 - minor usability problem
3 - major usability problem; important to fix
4 - usability catastrophe; imperative to fix

Issues Found

Difficult to find specific food items
Violates: Recognition rather than recall
Fix: list food items alphabetically
Severity - 2

On the Cart Display Screen the time in which the order can be cancelled is unavailable
Violates: “Visibility of system status“
Fix: add a timer below the X to let users know when the time for cancellation would expire
Severity - 2

Background color is solid white
Violates: Aesthetic and minimalist design
Fix: use gradient or pattern instead
Severity - 1

Picture of food items/meals unavailable hence users unsure of what is being purchased
Violates: Match between system and the real world
Fix: include pictures of food item/meals
Severity - 1

Spacing between words and size of words on the restaurant screen is different from the menu screen (screens are similar in layout)
Violates: “Consistency and Standards “
Fix: Standardize words size and spacing for similar screens
Severity - 1

On the Cart checkout page sub parts of main orders are listed in line with the main orders
Violates: “User Control and Freedom “
Fix: Add a tab space to sub parts of main orders
Severity - 1

Severity Rating

0 - don’t agree that this is a usability problem
1 - cosmetic problem
2 - minor usability problem
3 - major usability problem; important to fix
4 - usability catastrophe; imperative to fix
